# Vercel Deploy

To deploy this site to Vercel:

1. Link your repository to Vercel by going to [vercel.com](https://vercel.com)
2. Import your GitHub repository
3. Vercel will automatically detect the configuration and deploy your site
4. Future pushes to your repository will automatically trigger new deployments

The site is configured to serve the `docs/` folder as the public root directory via `vercel.json`.

# Custom Domain

To add a custom domain (glaum.ca):

1. Go to your project settings in the Vercel dashboard
2. Navigate to the "Domains" section
3. Add `glaum.ca` as a custom domain
4. Follow Vercel's instructions to configure your DNS records
5. Once DNS propagation is complete, your site will be accessible at glaum.ca
